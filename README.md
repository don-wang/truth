# truth


## Install Requirements

- [Python 2.7](https://www.python.org/)
- [virtualenv](https://virtualenv.pypa.io/)
- [pip](https://pip.pypa.io/)

## Deploy


```sh
$ git clone adress
$ cd folder
$ virtualenv flask
$ source flask/bin/activate
$ pip install -r requirements.txt
```

## Run

```sh
$ ./run.py
```
